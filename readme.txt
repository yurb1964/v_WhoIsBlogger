Добрый день,

Направляю вам тестовое задание.

Реализовано на: PostgreSQL (localhost), Python + библиотеки EasyGui, psycopg2, pandas, sys.

На localhost создается database YURB.

Представляю 2 модуля.

insert_test_data_cod.py - создание БД и загрузка тестовых данных.
Ограничение - не все исключения обрабатываются, поэтому при повторном запуске нужно убедится, что все сессии закрыты (если было падение).

run_sqls_cod.py - выполнение SQL запросов.
Вывод результатов сделан простой в STDOUT (на экран).

Тестовые данные:
users.txt
items.txt
purchases.txt

Данные частично сгенерированы через random().

Тестовые данные нужно положить в одну папку. При выполнении insert_test_data_cod.py - открывается окно с запросом папки.
